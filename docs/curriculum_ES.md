## Capítulo 1 — Reinicio del Sistema

### 📡 Historia

> El Comandante Orion despierta en una cabina oscura. Las luces de emergencia parpadean. La IA de la nave ha desaparecido. Recoges un terminal deteriorado y ves un cursor parpadeante. *"Cadete, ¿puedes escribir?"* Una sola línea de Python puede volver a encender las luces.

### Descripción

La nave del astronauta ha aterrizado de emergencia en un asteroide errante. Todos los sistemas están fuera de línea. El jugador debe escribir sus primeros scripts de Python para restaurar las funciones básicas de la nave.

### Temas de Python

| Tema | Descripción |
|------|-------------|
| `print()` | Mostrar texto en la consola — tu primera comunicación con la nave |
| Variables | Almacenar datos de misión: nombre del astronauta, porcentaje de oxígeno, nivel de combustible |
| Tipos de Datos | Enteros, flotantes, cadenas y booleanos — los cuatro pilares de los datos de la nave |
| Formato de Cadenas | f-strings para construir mensajes de estado dinámicos |
| Comentarios | Documentar el código para que futuros cadetes puedan entenderlo |

### Lecciones

| # | Título | Objetivo de Aprendizaje | XP |
|---|--------|------------------------|-----|
| 1 | ¡Hola, Espacio! | Usar `print()` para mostrar el primer mensaje de socorro del astronauta | 50 |
| 2 | Nombrando la Misión | Declarar variables para almacenar el nombre de la nave, tripulación y combustible | 50 |
| 3 | Datos en el Cosmos | Identificar y usar los tipos `int`, `float`, `str` y `bool` | 75 |
| 4 | Reportes de Estado | Construir mensajes legibles con f-strings y concatenación de cadenas | 75 |
| 5 | Anotar los Registros | Escribir comentarios significativos y entender la estructura del código | 50 |
